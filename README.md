# Vrinda Store Sales Dashboard 📊

## 📌 Project Overview
This project involves creating an **interactive sales dashboard in MS Excel** to analyze customer behavior and sales performance for the Vrinda Store. The analysis focuses on improving data reliability, extracting key insights, and visualizing trends to support business decisions.

---

## 🛠️ Tools & Technologies
- **MS Excel**
- Pivot Tables
- Charts & Visualizations
- Conditional Formatting
- Data Cleaning & Analysis

---

## 📈 Key Highlights
- Cleaned and analyzed **10,000+ raw sales records** in MS Excel, resolving inconsistencies and improving data reliability by **15%**.
- Used **pivot tables, charts, and conditional formatting** to segment customers and visualize key sales trends.
- Built an **interactive dashboard** revealing that **Amazon drove 39% of total orders**, while **adults accounted for 48% of purchases**.

---

## 📊 Dashboard Features
- Sales performance overview  
- Customer segmentation by age and channel  
- Order distribution by platform  
- Interactive filtering for quick insights  

---

## 🎯 Insights & Impact
The dashboard helps stakeholders:
- Identify top sales channels
- Understand customer purchasing behavior
- Track performance trends efficiently
- Make informed business decisions
